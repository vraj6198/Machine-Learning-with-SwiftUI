

# How to Install Project

### Install Cocoapods

```ruby
sudo gem install cocoapods
```

### Create podfile
```ruby
pod init
```

### Now, Open Podfile
```ruby
open -a Xcode podfile
```

#### Install pod
```ruby
pod install
```

### ML Kit libraries in podfile
```ruby
  pod 'Firebase/Core'
  pod 'GoogleMLKit/ImageLabelingAutoML'
  pod 'GoogleMLKit/LinkFirebase'
```
### import Firebase and MLKit in Project File

``` swift
import Firebase
import MLKit
```
